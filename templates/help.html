Frequently Asked Questions

What is the page when I first signed up or logged in?
  • When you login for the first time it will give you the select text webpage.
  • The left column shows you the text that is previously uploaded by you.
  • If you logged in for the first time then select text portion will be empty till you upload a file in it.
  • You can upload a text from the right colum which shows you the upload text.
    • How to upload a new text?
  • If you are logged in or signed in for the first time you are on the correct page.
  • If you want to add new file on top of other please click on the select text on the navigation bar.
  • Here you can click on th choose file which is on the upload text column.
  • You can only select one file at a time and no more than that so choose a file and select ok.
  • Once you see that file name next ti the choose file button click upload and that will start uploading a file for you.

  • If you have to upload a new work then there is a column on the right which shows upload text.

    • Where can I see my information?
  • My account page is the source where you can see your information and what file you are currently working on.
  • You can track your last signed out date and what line of the file you are on and what is the chapter that you are currenty working on.
  • You also have the your work column which stores all your previous work and files for which you do not have to go the loader page to open the previously uploaded files.
    • How does the My Account works?
  • On the left top colum you have all the files that you have worked on or working on.
  • You cannot add new file from here but you can open the files that were uploaded earlier.

  • On the left bottom column you can see the most recent work that you have worked on.
  • It shows you your most recent work on the file including text, chapter, index, sentence, phrase and color.

  • On the right side column you can see your user-name, password and last signed out date.
  • You can update your user-name and password by clicking edit next to your user-name and password
    • How can I change my username?
  • If you are searching to change your username then you can go to the my account page that is on the top right side of the navigation bar.
  • On the right column you will see my information and your username in there.
  • To change it you just need to click edit and it will give you a box to input new username.
  • Once you put your new username click on ok and then it will give you a pop-up alert showing that username updated successfully and you are all set with the new username.
    • How can I change my Password?
  • If you are searching to change your password then you can go to the my account page that is on the top right side of the navigation bar.
  • On the right column you will see my information and password section in there (You cannot see you old password because of the privacy of your data but once you are logged in you can change the password from there.)
  • To change it you just need to click edit and it will give you a box to input new password.
  • Once you put your new password click on ok and then it will give you a pop-up alert showing that password updated successfully and you are all set with the new password.
